## Getting Started with Fresh Framework
Fresh is yet another full stack framework that has recently hit the market. 
Fresh framework is built for speed, reliability, and simplicity and supported by the Deno company that also owns Deno CLI. Fresh is Preact based and runs over Deno.


| Key Features                | 
| :--------------------- |
| `Edge rendering`: Just-in-time rendering on the edge|
| `Island of Components`: Island based client hydration for maximum interactivity|
| `Zero JS `: Zero runtime overhead: no JS is shipped to the client by default. |
| `Bulding steps`: No build step. |
| `Configuration`: No configuration necessary. |
| `TypeScript`: TypeScript support out of the box.        

For more visit: https://fresh.deno.dev
